# Pro Docx Gen v1.6.2

专业 Word 文档生成技能，支持修订追踪、批注、格式保留，商务级排版。

## 功能亮点
- 📝 **商务级排版**：auto_style 引擎，行内加粗 + 表格加粗精准控制
- 🔍 **修订追踪 + 批注**：协作编辑全流程支持
- 📄 **格式保留**：原样提取/修改 Word 内容，不丢样式
- 🌐 **多语言**：中文 PDF → 英文 DOCX 结构化转换
- 🎯 **企业级文档**：培训手册/产品规格书/竞品分析报告均可胜任
